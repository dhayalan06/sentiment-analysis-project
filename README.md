# 😊 Sentiment Analysis Project

## 📌 Project Overview

This project focuses on analyzing textual data to determine whether the sentiment expressed in the text is **positive, negative, or neutral**. Sentiment Analysis is a Natural Language Processing (NLP) technique used to understand people's opinions, emotions, and attitudes from text data.

The analysis can be applied to various sources such as **Amazon product reviews, social media posts, and news articles** to understand public opinion and sentiment trends.

---

# 🎯 Objectives

The main objectives of this project are:

* Analyze text data to classify sentiment as **positive, negative, or neutral**
* Apply **Natural Language Processing (NLP)** techniques for text analysis
* Detect emotions and opinions using **lexicon-based approaches**
* Understand **public opinion and trends** from textual data
* Use insights to support **marketing strategies, product improvements, and social analysis**

# 🔍 Sentiment Analysis Process

The project follows these steps:

1. Load text dataset using **Pandas**
2. Process text using **TextBlob NLP**
3. Analyze sentiment polarity
4. Classify text as **Positive, Negative, or Neutral**
5. Visualize sentiment results using charts
